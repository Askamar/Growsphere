# GrowSphere 🌐📈

**GrowSphere** is a modern, premium web application designed to help users manage their financial portfolio, set financial goals, and plan Systematic Investment Plans (SIP). The project boasts a high-end dynamic user interface, leveraging smooth animations, glassmorphism aesthetics, and responsive layout designs. 

## ✨ Key Features
- **📊 Portfolio Management**: Track and analyze your investments effectively.
- **🎯 Goals Tracking**: Set, manage, and track individual financial milestones.
- **📅 SIP Planner**: Plan your Systematic Investment Plans with an intuitive UI.
- **📚 Learn/Onboarding**: Informational onboarding flow and glossary for user education.
- **💎 Premium Design**: Fully built with standard CSS incorporating vibrant color gradients, modern typography, glassmorphism styling, and responsive micro-interactions.

## 🛠️ Technology Stack
- **Frontend Framework**: React 18, React Router v6
- **Build Tool**: Vite
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Modern customized design system)
- **Data Validation**: Zod

## 🚀 Getting Started

Follow these steps to set up the project locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Askamar/Growsphere.git
   cd Growsphere
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the local development server:**
   ```bash
   npm run dev
   ```

4. **Build for Production:**
   ```bash
   npm run build
   ```

## 📂 Project Structure
- `src/pages/` - Contains the primary route views (Home, Portfolio, Goals, SIPPlanner, Learn, Onboarding, Explore).
- `src/shared/` - Contains reusable UI components and shared utilities.
- `src/index.css` - Custom styling library incorporating the overall glassmorphism design system.
